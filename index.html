About Blind Code


Blind Code is a not-for-profit coding platform, where you have to write code for a problem. The difference is, you can't see the code. It is a programming community that fosters learning and friendly competition. We host featured contests every month.
We also provide platform to practice "Blind Coding" which definitely helps the coder to improve his coding skills


Our Values


MISSION: The BlindCode team is on a mission to make every developer in the world confident enough to write code without any hasitation and help them increase their speed and efficiency of writing code.
